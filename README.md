# Portofolio Web Modern

Ini adalah template portofolio web modern yang dibangun dengan HTML5, Tailwind CSS, dan sedikit JavaScript. Template ini dirancang untuk menjadi responsif, cepat, dan mudah disesuaikan.

## Fitur
- **Desain Responsif**: Terlihat bagus di perangkat apa pun (desktop, tablet, mobile).
- **Smooth Scrolling**: Navigasi antar bagian dengan efek gulir yang halus.
- **Interaktif**: Termasuk menu mobile dan simulasi formulir kontak.
- **Tailwind CSS**: Menggunakan Tailwind via CDN untuk kemudahan penyesuaian gaya tanpa perlu build step yang rumit.
- **Font Awesome**: Ikon media sosial dan teknis yang lengkap.

## Cara Kustomisasi
1. Buka `index.html`.
2. Cari placeholder seperti `Nama Anda`, `email@contoh.com`, dan lainnya, lalu ganti dengan informasi Anda sendiri.
3. Ganti gambar profil dan gambar proyek di folder `assets/` atau gunakan URL gambar langsung di `index.html`.
4. Sesuaikan bagian `Keahlian`, `Proyek`, dan `Pengalaman` sesuai dengan portofolio asli Anda.

## Cara Hosting di GitHub Pages
1. Push kode ini ke repositori GitHub Anda.
2. Di GitHub, buka tab **Settings** repositori Anda.
3. Pilih menu **Pages** di bilah sisi kiri.
4. Di bawah bagian **Build and deployment**, pastikan Source diatur ke "Deploy from a branch".
5. Pilih branch `main` (atau branch utama Anda) dan folder `/ (root)`.
6. Klik **Save**.
7. Tunggu beberapa menit, dan situs Anda akan tersedia di `https://username.github.io/nama-repo/`.
